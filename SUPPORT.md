# LocaleShot Beta Support

Use GitHub Issues in this repository for:

- Install problems
- Simulator attach problems
- Missed or incorrect replayed clicks
- Language generation failures
- Screenshot output or review gallery issues

When reporting a bug, include:

- macOS version
- Xcode version
- Simulator device and iOS runtime
- LocaleShot version
- What you expected
- What happened

Do not attach private app builds or confidential screenshots unless you are comfortable sharing them publicly.
